# cryoDRGN J1442 latent landscape (z10)

PCA/UMAP visualization of the 10-D latent space from the J1442 z10 run
(`results_cryodrgn/J1442_real/train_z10/z.100.pkl`, 230,396 particles).
Generated by `scripts/cryodrgn/cryodrgn_landscape.py`.

## Key metrics

- PC1 explains 15.5% of standardized latent variance; PC2 explains 11.9%.
- K=3 GMM: min component separation 1.90 SD, mean 2.30 SD.
  (below 2 SD threshold for discrete wells; continuous landscape with preferred states)
- PC1 marginal shows 3 peaks at ~−1.3, 0, +1.3 with populated valleys between them.
- K=8 fine-grain: all 8 ellipses tile the same 3 density peaks; no hidden 4th/5th state.
- UMAP projection confirms 3 density regions with diffuse boundaries.

## GMM peak z-vectors (`z_gmm_peaks.txt`)

Three rows, each 10 values: the GMM component means in the 10-D latent space.
These are the "centers" of the three preferred conformational states.

**Decode with `cryodrgn eval_vol` (requires GPU):**
```bash
cryodrgn eval_vol results_cryodrgn/J1442_real/train_z10/weights.pkl \
  --config results_cryodrgn/J1442_real/train_z10/config.yaml \
  --zfile results_cryodrgn/J1442_real/landscape_z10/z_gmm_peaks.txt \
  --Apix 0.83 \
  -o results_cryodrgn/J1442_real/landscape_z10/peak_volumes
```
Output: `vol_000.mrc` (P7), `vol_001.mrc` (P6), `vol_002.mrc` (P8).
Run on a CUDA-capable node — eval_vol uses the neural network decoder.

## `peak_volumes/`

Pre-decoded peak volumes (if present): mrc files at Apix=0.83 Å/px for the
three GMM component means. Compare to CryoSPARC NU-Refinement maps to verify
the decoder learned real structural density.
